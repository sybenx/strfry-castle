A note on private messages: gift-wrapped DMs (NIP-59) are signed by random one-time keys, so this relay cannot tell the owner's own messages from a stranger's. All gift wraps age out at the next purge like any other unprotected event. For permanent, private message storage, use a dedicated relay built for that purpose.
